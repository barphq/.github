### How we prioritise severity (CVSS)
We assess severity using CVSS (Common Vulnerability Scoring System) where applicable.  
If you donâ€™t know CVSS, you can still help us by choosing one of these:

- **Critical**: remote takeover, auth bypass, admin compromise, large-scale data exposure
- **High**: access to sensitive data, privilege escalation, payment/identity impact
- **Medium**: limited data exposure, meaningful abuse with constraints
- **Low**: minor issues, unlikely exploitation, minimal impact

If a CVE assignment is appropriate, we will coordinate disclosure with the reporter and relevant maintainers.

### Please avoid
- Public disclosure before coordinated disclosure
- Scanning that degrades service availability
- Sending sensitive personal data to demonstrate impact
